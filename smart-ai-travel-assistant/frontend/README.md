# Smart AI Travel Assistant

This is a modern travel planning website that helps users explore destinations and design personalized journeys.

## Features
- AI travel recommendations
- Beautiful travel UI
- Destination explorer
- Smart itinerary planning

## Tech Stack
- Next.js
- React
- Tailwind CSS

## Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
